Metodos.

DiscoveryFull

APIDescription
POST api/DiscoveryFull/CadastraPesquisa_PFPJ

Cadastro de pesquisa de pessoa física / Jurídica - Este método permite o cadastro de uma pesquisa por pessoa física / Jurídica.
POST api/DiscoveryFull/CadastraPesquisa_Adv

Cadastro de pesquisa de Advogados - Este método permite o cadastro de uma pesquisa por advogados.
POST api/DiscoveryFull/CadastraPesquisa_Assuntos

Cadastro de pesquisa de Assuntos / Tópicos - Este método permite o cadastro de uma pesquisa por assuntos / Tópicos.
POST api/DiscoveryFull/CadastraPesquisa_NumProcessos

Cadastro de pesquisa do tipo "Lista de processos" - Este método permite o cadastro de uma pesquisa por números de processo.
POST api/DiscoveryFull/CadastraPesquisa_Preditiva

Cadastro de pesquisa preditiva - Este método permite o cadastro de uma pesquisa preditiva.
POST api/DiscoveryFull/buscaStatusPesquisa

Busca o status das pesquisas cadastradas - Este método retorna o status das pesquisas cadastradas anteriormente.
POST api/DiscoveryFull/buscaHistoricoPesquisas

Busca o histórico das pesquisas cadastradas - Este método retorna o histórico das pesquisas cadastradas em um período de tempo.
POST api/DiscoveryFull/buscaDadosResultadoPesquisa

Busca o resultado da pesquisa - Este método retorna todos os dados e processos encontrados na pesquisa solicitada.
POST api/DiscoveryFull/buscaDadosResultadoPesquisa_Count

Busca o resultado da pesquisa count - Este método retorna todos os dados da pesquisa count solicitada.
POST api/DiscoveryFull/autenticaAPI

Gera token para utilizar demais metodos da API
POST api/DiscoveryFull/buscaDadosMovimentoPesquisa

Retorna os andamentos dos processos encontrados na pesquisa - Este método retorna todos os andamentos dos processos encontrados na pesquisa solicitada.
POST api/DiscoveryFull/buscaDadosPublicacaoPesquisa

Retorna as publicações dos processos encontrados na pesquisa - Este método retorna as publicações dos processos encontrados na pesquisa solicitada.
POST api/DiscoveryFull/buscaStatusDocumentoProcesso

Busca o status dos documentos do processo - Este método retorna o status dos documentos dos processos.
POST api/DiscoveryFull/buscaDadosDocIniciaisPesquisa

Retorna os documentos iniciais dos processos encontrados na pesquisa - Este método retorna os documentos dos processos encontrados na pesquisa solicitada.
POST api/DiscoveryFull/buscaStatusPesquisa_Preditiva

Busca o status das pesquisas preditivas cadastradas - Este método retorna o status das pesquisas preditivas cadastradas anteriormente.
POST api/DiscoveryFull/buscaDadosResultadoPesquisa_Preditiva

Busca o resultado da pesquisa preditiva- Este método retorna todos os dados e processos encontrados na pesquisa preditiva solicitada.
POST api/DiscoveryFull/confirmaRecebimentoPublicacaoPesquisa

Confirma o recebimento das publicações dos processos encontrados na pesquisa - Este método confirma o recebimento das publicações dos processos encontrados na pesquisa solicitada.
POST api/DiscoveryFull/confirmaRecebimentoDocIniciaisPesquisa

Confirma o recebimento dos documentos iniciais dos processos encontrados na pesquisa - Este método confirma o recebimento dos documentos iniciais dos processos encontrados na pesquisa solicitada.
